# SQL Scripts - Voting Management System

This folder contains all SQL scripts used to design and implement the Voting Management System database.

## 📂 Files Included

### 1. schema.sql
- Contains CREATE TABLE statements
- Defines database structure including:
  - VOTER
  - CANDIDATE
  - PARTY
  - CONSTITUENCY
  - ADMIN
  - VOTE
- Includes primary keys and foreign key relationships

---

### 2. insert.sql
- Contains INSERT statements
- Used to populate tables with sample data
- Includes voters, candidates, parties, and votes

---

### 3. queries.sql
- Contains SQL queries used for operations such as:
  - JOIN operations
  - Aggregation (COUNT of votes)
  - DELETE operations
  - VIEW creation and retrieval

## 🎯 Purpose

These scripts demonstrate:
- Database design principles
- Data manipulation using SQL
- Query execution for real-world voting system operations
