# Query Outputs - Voting Management System

This folder contains the output screenshots of SQL queries executed for the Voting Management System project.

## 📌 Contents

The outputs demonstrate the working of SQL queries including:

- Joining Voter and Constituency tables
- Counting total votes per candidate
- Deleting voter records and verifying updates
- Displaying data using database views

## 📷 Files Included

- voter-constituency-join.png → Displays voter details with constituency name
- vote-count-result.png → Shows total votes received by each candidate
- voter-table-after-delete.png → Shows updated voter table after deletion
- candidate-view-output.png → Displays candidate view (name, age, gender)
- total-voters-view.png → Displays total number of voters

## 🎯 Purpose

These outputs verify that:
- SQL queries are working correctly
- Database relationships are properly implemented
- Views and aggregate functions are functioning as expected
